class darwin.networks.EchoStateNetwork(n_inputs, n_neurons, sparsity, spectral_radius, readout_layers, readout_optimizer, readout_loss, readout_metrics, nbit_quant=None, seed=0)¶

Echo State Network with no feedback weights and no teacher signal. Dynamic systems simulations should not leverage this model. Classification and regression tasks can use multi-level fully-connected layers as a readout in order to solve complex prediction tasks.

Parameters

  • n_inputs (int) – The number of input values.

  • n_neurons (int) – Value indicating how many neurons have to be used in the hidden layer.

  • sparsity (float) – Percentage of connections to be removed in the hidden layer.

  • spectral_radius (float) – Adjusting factor for hidden neuronal connections.

  • readout_layers (dict) – Collection of dictionaries indicating the structure.and parameters of dense layers used for the readout implementation.

  • readout_optimizer (keras.optimizers) – Function that belongs to the Keras optimizer library.

  • readout_loss (keras.losses) – Function that belongs to the Keras loss functions library.

  • readout_metrics (keras.metrics) – Function that belongs to the Keras metrics library.

  • nbit_quant (int, defaults to None) – Number of bits used for the quantization. None indicates full precision will be used.

  • seed (int, optional) – Initial value for the random number generator.

class darwin.layers.readout.MLDense(n_inputs, readout_layers, readout_optimizer, readout_loss, readout_metrics, nbit_quant)¶

Implementation of a readout layer consisting in a multi-layer fully-connected classifier or regressor. This class is meant to be used only in combination with objects derived from one of the darwin.networks classes.

Parameters

  • n_inputs (int) – The number of input values for the readout layer

  • readout_layers (dict) – Collection of dictionaries indicating the structure and parameters of dense layers used for the readout implementation

  • readout_optimizer (keras.optimizers) – Function that belongs to the Keras optimizer library

  • readout_loss (keras.losses) – Function that belongs to the Keras loss functions library

  • readout_metrics (keras.metrics) – Function that belongs to the Keras metrics library

  • nbit_quant (int) – Number of bits used for the quantization. None indicates full precision will be used
